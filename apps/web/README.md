# TextPulse Web

Next.js frontend for TextPulse AI.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Environment
- `NEXT_PUBLIC_API_BASE_URL`: public URL of the FastAPI backend

If this variable is missing, the app falls back to a built-in demo workspace so design and UX review can continue without a live backend.
